# test-task-NatVor

## Install

### Clone this repository on your Docker host, cd into test directory and run compose up:

```
git clone git@github.com:NatVor/test-task-NatVor.git
cd test-task-NatVor
docker-compose up -d
```

## Prerequisites:

* Docker
* Docker Compose 
* Ansible
* Terraform
* Azure CLI
